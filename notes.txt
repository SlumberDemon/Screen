
Hand = object that stores state about the cursor/touch

Canvas = client's whole screen (0 to max pixel)
View = client's whole screen (0.0 to 1.0)

CanvasPosition = a place in the canvas (0 to max pixel)
ViewPosition = a place in the view (0.0 to 1.0)

Corners = four view positions (from 0.0 to 1.0) that show where the corners of a quadrilateral are

Hex = a hex string
Colour = an object with a hex and list of screens
Screen = a positioned colour
World = a single screen that covers the whole view
